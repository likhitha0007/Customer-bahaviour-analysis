# Customer Behaviour Analysis

This repository contains a Django-based project for Customer Behaviour Analysis using data mining techniques and AI-driven recommendations.

Quick start

- Requirements: Python 3.8+ and the packages in `requirements.txt`.
- Install dependencies:

```bash
pip install -r requirements.txt
```

- Apply migrations and run the development server:

```bash
python manage.py migrate
python manage.py runserver
```

Notes

- Large datasets and model artifacts are excluded via `.gitignore`. Add your datasets to the `datasets/` directory locally if needed.
